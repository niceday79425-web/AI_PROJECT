Um Segundo Salário para Trabalhadores: Conceitos Essenciais do Calendário de Dividendos

Para usar o calendário de dividendos corretamente, é fundamental entender o significado preciso de cada data. Veja os 3 conceitos que mais geram confusão.

01 O Significado Exato da Data Ex-Dividendo

Muitos investidores sabem que "é preciso comprar antes da Data Ex-Dividendo para receber o dividendo" — e isso está correto. A Data Ex-Dividendo (Ex-Dividend Date) é o dia a partir do qual quem comprar a ação não terá mais direito ao próximo dividendo. Quem compra na data ex ou depois fica de fora do pagamento.

📅 Exemplo de Liquidação
Compra 1 dia antes da data ex
Liquidação → na data ex
✅ Recebe dividendo
Compra na data ex
Liquidação → após a data ex
❌ Sem direito ao dividendo

Nos EUA, o ciclo de liquidação é T+1 (desde 2024), ou seja, a compra é registrada no dia seguinte. Portanto, basta comprar até o dia útil anterior à data ex para ter direito ao dividendo. Para otimização fiscal, porém, o ideal é manter as ações por um período mais longo do que apenas capturar dividendos.

02 Diferença Entre Data de Declaração, Data de Registro e Data de Pagamento

📢 Data de Declaração
O conselho de administração anuncia oficialmente o pagamento do dividendo. O valor e as demais datas são definidos aqui — é o primeiro dado que o investidor deve verificar.
📋 Data de Registro (Record Date)
Data em que a empresa confirma quais acionistas têm direito ao dividendo. Definida após a data ex. Somente quem consta no registro nessa data recebe o pagamento.
💰 Data de Pagamento
Data em que o dividendo é efetivamente creditado na conta do investidor (Payment Date). Geralmente ocorre de 2 a 4 semanas após a Data de Registro.
💡 Ordem resumida: Data de Declaração → Data Ex-Dividendo → Data de Registro → Data de Pagamento

03 Montando um Portfólio de Dividendos Mensais com Ações Trimestrais

Mesmo sem ETFs de dividendos mensais, é possível combinar 3 grupos de ações trimestrais com meses de pagamento diferentes para receber dividendos todo mês. A maioria das empresas americanas paga dividendos a cada 3 meses, mas em meses distintos.

Meses de Pagamento Exemplos de Empresas Setor
Jan / Abr / Jul / Out JNJ, JPM, BAC Saúde / Financeiro
Fev / Mai / Ago / Nov MSFT, AAPL, KO Tecnologia / Consumo Básico
Mar / Jun / Set / Dez XOM, CVX, PG Energia / Bens de Consumo

Com pelo menos uma ação de cada grupo, você recebe dividendos todos os meses. Mas lembre-se: ao montar o portfólio, priorize a qualidade da empresa (payout ratio, FCF, taxa de crescimento de dividendos), e não apenas o mês de pagamento.

💡 Dica: Use o Rastreador de Dividendos para filtrar as melhores ações de cada grupo e verificar os meses de pagamento.
