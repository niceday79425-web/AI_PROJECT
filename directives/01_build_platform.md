# Directive: 미국 주식 월 배당 복리 계산기 & 자동화 블로그 구축

이 지침서는 미국 주식 배당 복리 계산기 서비스와 이를 홍보/운영하기 위한 자동화된 노데이터베이스(No-DB) 기반 블로그 플랫폼 구축 절차를 정의합니다.

## 1. 프로젝트 개요 (Overview)

*   **주제**: 미국 주식 월 배당 복리 계산기 & 자동화 블로그
*   **목표**: 사용자에게 직관적인 웹 기반 복리 계산 도구를 제공하고, 분석된 데이터를 바탕으로 블로그 콘텐츠를 자동 생성하여 트래픽을 유도함.
*   **기술 스택**:
    *   **프론트엔드**: HTML5, CSS3, Vanilla JavaScript
    *   **시각화**: Chart.js (스택형 막대그래프)
    *   **자동화**: Python (데이터 수집 및 블로그 HTML/JSON 생성)
    *   **배포**: Cloudflare Pages (정적 호스팅)

## 2. 상세 구현 단계 (Step-by-Step)

### Step 1: 프로젝트 구조 및 기반 작업
프리미엄 핀테크 느낌의 디자인과 정적 호스팅에 최적화된 파일 아키텍처를 구성합니다.
*   `index.html`: 메인 페이지 (계산기 인터페이스 + 최신 투자 정보 리스트)
*   `css/style.css`: 핀테크 스타일의 다크 모드 디자인 (고급스러운 보라색/파란색 그라디언트 활용, 모바일 대응)
*   `js/calculator.js`: 복리 계산 로직 및 Chart.js 연동
*   `blog/`: 생성된 블로그 포스트 HTML 파일 저장소
*   `posts.json`: 블로그 글 목록 메타데이터 관리 (제목, 날짜, 링크, 요약)

### Step 2: 계산기 로직 (Calculator Logic) 구현
사용자 입력에 따른 자산 성장 시뮬레이션을 수행합니다.
*   **입력값**: 티커(Ticker), 초기 투자금, 월 적립액, 연 배당률, 연 주가 상승률, 투자 기간(년)
*   **핵심 기능**: 
    - 1개월 단위 자산 증가 추이 계산
    - 배당 재투자(DRIP) 및 배당소득세(15%) 반영
    - '계산하기' 버튼 클릭 시 실시간 시뮬레이션 결과 도출
*   **결과값**: 최종 자산, 총 배당금, 원금 대비 수익률
*   **시각화**: Chart.js를 사용해 [원금], [평가 수익], [배당 재투자 수익]을 스택형 막대그래프로 표현

### Step 3: 블로그 시스템 (No-DB Structure) 구축
별도의 DB 없이 JSON 파일을 데이터베이스처럼 활용합니다.
*   `posts.json` 파일을 통해 블로그 글 목록을 관리.
*   메인 페이지 로딩 시 JavaScript가 `posts.json`을 읽어 하단 '최신 투자 정보' 섹션에 동적으로 리스트 생성.
*   Python 스크립트가 새 분석글을 작성할 때마다 `blog/*.html` 파일 생성 및 `posts.json` 업데이트.

## 3. 실행 계획 (Action Items)

1.  **Directive 작성 및 승인**: 본 파일(`01_build_platform.md`)을 통해 전체 로드맵 확정. (현재 단계)
2.  **프론트엔드 기반 구축**: `index.html`, `css/style.css` 작성 (고급스러운 UI 우선 구현).
3.  **계산 엔진 구현**: `js/calculator.js` 작성 및 Chart.js 연동 테스트.
4.  **자동화 스크립트 작성**: `execution/` 폴더에 데이터 수집 및 블로그 생성 Python 코드 작성.
5.  **연동 및 테스트**: 계산기 기능 확인 및 블로그 리스트 동적 로딩 확인.
